    下载安装node.js+cnpm+hexo

    环境：W10

    官网：https://nodejs.org/zh-cn/ （下载了LTS版本）

    Win+R输入cmd打开命令提示符依次输入

    node -v #查看node版本 
    npm -v  #查看npm版本  
    npm install -g cnpm --registry=http://registry.npm.taobao.org #安装淘宝的cnpm 管理器
cnpm -v #查看cnpm版本 
cnpm install -g hexo-cli #安装hexo框架
hexo -v #查看hexo版本 
mkdir XiaoPanTongXue #创建XiaoPanTongXue目录
cd XiaoPanTongXue #进入XiaoPanTongXue目录
sudo hexo init #生成博客 初始化博客
hexo s  #启动本地博客服务
http://localhost:4000/ #本地访问地址
hexo n "我的博客生成记录" #创建新的文章

    发布到GitHub

    如果用gitee的话要在对应仓库开启 Gitee Pages 服务

    在github注册自己的账号记住自己的邮箱账号和密码，因为各种网站各种账号密码，我记忆已经混乱，都记录在自己手机的记事本上。（主要是一些密码要求的格式比较复杂，还要大小写特殊字符什么的就容易忘）

    下载安装git：https://git-scm.com/downloads/

    一路默认，安装完成后，打开文件夹C:\Users\Administrator.ssh（Administrator是当前用户名），在空白处点鼠标右键选择“Git Bush Here” ，打开gitbush。

    配置用户名和邮箱：

    git config --global user.name “用户名”
git config --global user.email “邮箱”
执行生成公钥和私钥的命令：
ssh-keygen -t rsa #按照提示回车3下（会在.ssh文件夹里面生成一个私钥 id_rsa和一个公钥id_rsa.pub。）
# 执行查看公钥的命令：
# cat ~/.ssh/id_rsa.pub

    到github上去添加公钥

    起一个公钥title把id_rsa.pub文件里的公钥复制到key里Add SSH key

    *加公钥就不需要每一次在博客hexo deploy的时候输入github的账号密码了，也防止了没有秘钥的人deploy，安全一点

    注意创建仓库时仓库名称

    xiaopantongxue816.github.io 黑体部分要和github用户名一致，不然访问不到

cnpm install --save hexo-deployer-git

    参考原文： https://yelog.org/2017/03/23/3-hexo-instruction/

    添加分类及标签

    在博客文件夹下命令行执行

    hexo new page categories

    成功提示

    INFO  Created: e：/XiaoPanTongXue/source/categories/index.md

    根据目录找到index.md文件打开默认是：

    title: categories
date: 2020-08-28 13:02:09

    在后面加上一行

    type: "categories"

    保存关闭。

    打开需要分类的文章添加categories属性，例如

    title: 博客记录
date: 2020-08-16 10:12:34
categories:
- 其它
- hexo

    其它是一级分类，hexo为二级分类，依次递减。

    添加标签的步骤类似

    hexo new page tags

    title: &#123;&#123; title &#125;&#125;
date: &#123;&#123; date &#125;&#125;
tags:
categories:

    置顶文章

    使用 top 将会置顶文章，多个置顶文章时，top 定义的值越大，越靠前。

    ---
title: 每天一个 linux 命令
date: 2017-01-23 11:41:48
top: 1
categories:
- 运维
tags:
- linux 命令
---
