    protobuf 三个关键字required、optional、repeated的理解

    required关键字

    必须，数据发送方和接收方都必须处理这个字段

    optional关键字

    可选，另外加了一个bool的变量，标记这个字段是否有值

    发送方在发送的时候，如果这个字段有值，为true，否则为false

    接收方在收到这个字段时，也会收到发送方同时发送的bool变量，根据变量判断这个字段是否有值

    repeated关键字

    重复，另外加了一个count计数变量，用于标明这个字段有多少

    发送方发送的时候，同时发送了count计数变量和这个字段的起始地址

    接收方在接受到数据之后，按照count来解析对应的数据。

    Demo

    1.下载之后，解压到本地，然后配置环境变量

    D:\Program Files\protoc-3.13.0-win64\bin

    2.准备一个User.proto文件，内容如下： 

    # 我们定一个User类，下面有4个属性，分别是id,name,age,mobile
option java_outer_classname="UserProtoBuf";  //生成java类名
option java_package="com.pjy.serial";        //对应的包名package
message User&#123;                        //message关键字定义了User对象，最后定义了四个属性字段
    required int32 id=1;
    required string name=2;
    optional int32 age=3;
    optional string mobile=4;
&#125;

    3.在命令行下通过如下命令，生成一个Java类（UserProtoBuf.java）

    D:\EclipseWorkspace\ProtobufDemo\src\main\java>protoc --java_out=. com\pjy\serial\User.proto

    4.在maven工程中加入protobuf-java依赖（要使用protoc创建的java类，必须要加入protobuf-java这个依赖 ）

    ！！！注意version要和protoc.exe版本一致，否则会报错

    <dependency>
      <groupId>com.google.protobuf</groupId>
      <artifactId>protobuf-java</artifactId>
      <version>3.13.0</version>
</dependency>

    idea 插件的使用 (GenProtobuf)

    搜索Genprotobuf插件下载，在main目录下新建proto目录

    右击proto目录 Mark Directory As ->Resouces Root,把编写好的JetProtos.proto文件放在该目录下

    pjysyntax = "proto3"; // PB协议版本

import "google/protobuf/any.proto"; // 引用外部的message，可以是本地的，也可以是此处比较特殊的 Any

package pjy.protobuf; // 包名，其他 proto 在引用此 proto 的时候，就可以使用 test.protobuf.PersonTest 来使用，
// 注意：和下面的 java_package 是两种易混淆概念，同时定义的时候，java_package 具有较高的优先级

option java_package = "com.pjy.serial"; // 生成类的包名，注意：会在指定路径下按照该包名的定义来生成文件夹
option java_outer_classname="PersonTestProtos"; // 生成类的类名，注意：下划线的命名会在编译的时候被自动改为驼峰命名

message PersonTest &#123;  
    int32 id = 1; // int 类型  
    string name = 2; // string 类型  
    string email = 3;  
    Sex sex = 4; // 枚举类型  
    repeated PhoneNumber phone = 5; // 引用下面定义的 PhoneNumber 类型的 message  
    map<string, string> tags = 6; // map 类型  
    repeated google.protobuf.Any details = 7; // 使用 google 的 any 类型  

    // 定义一个枚举  
    enum Sex &#123;      
        DEFAULT = 0;      
        MALE = 1;      
        Female = 2;  
    &#125;  

    // 定义一个 message  
    message PhoneNumber &#123;    
        string number = 1;    
        PhoneType type = 2;    

        enum PhoneType &#123;      
            MOBILE = 0;      
            HOME = 1;      
            WORK = 2;
